## Función principal
Verifica si el numero es par o impar y retorna un booleano que indica (true == par) o (false == impar).

~~~ C (lenguaje en el que esta escrito)
bool CalcularNumerosPares(int numero)
{
    if (numero % 2 == 0) 
      //el numero es par
    {
        return true;
    }
  
  else
    //caso contrario es impar
  {
     return false;
  }
}
~~~
